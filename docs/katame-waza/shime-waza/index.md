# Shime-waza — 絞技

Las **shime-waza** (técnicas de estrangulación) buscan comprimir las arterias carótidas (estrangulación vascular) o la tráquea (estrangulación respiratoria) para provocar la rendición o la pérdida de consciencia del uke. El Kodokan reconoce **12 técnicas**.

!!! warning "Seguridad"
    Las shime-waza son técnicas de alto riesgo. Deben practicarse siempre bajo supervisión de un instructor calificado y con plena atención a la señal de rendición del uke.
