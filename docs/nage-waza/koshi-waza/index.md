# Koshi-waza — 腰技

Las **koshi-waza** (técnicas de cadera) son proyecciones en las que la cadera actúa como punto de apoyo o palanca para elevar y lanzar al oponente. El Kodokan reconoce **10 técnicas**.

| Técnica | Kanji | Traducción |
|---------|-------|------------|
| [Uki-goshi](uki-goshi.md) | 浮腰 | Cadera flotante |
| [O-goshi](o-goshi.md) | 大腰 | Gran cadera |
| [Koshi-guruma](koshi-guruma.md) | 腰車 | Rueda de cadera |
| [Tsurikomi-goshi](tsurikomi-goshi.md) | 釣込腰 | Cadera de tirón y empuje |
| [Sode-tsurikomi-goshi](sode-tsurikomi-goshi.md) | 袖釣込腰 | Cadera de tirón y empuje de manga |
| [Harai-goshi](harai-goshi.md) | 払腰 | Cadera barredora |
| [Tsuri-goshi](tsuri-goshi.md) | 釣腰 | Cadera colgante |
| [Hane-goshi](hane-goshi.md) | 跳腰 | Cadera de salto |
| [Utsuri-goshi](utsuri-goshi.md) | 移腰 | Cadera de transferencia |
| [Ushiro-goshi](ushiro-goshi.md) | 後腰 | Cadera trasera |
