# Ashi-waza — 足技

Las **ashi-waza** (técnicas de pierna) son proyecciones en las que los pies o piernas ejecutan la acción principal: barrer, bloquear o enganchar la pierna del oponente. Es la familia más numerosa del judo de pie. El Kodokan reconoce **21 técnicas**.
