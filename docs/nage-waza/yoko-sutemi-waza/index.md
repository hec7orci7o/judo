# Yoko-sutemi-waza — 横捨身技

Las **yoko-sutemi-waza** (técnicas de sacrificio lateral) son proyecciones en las que el tori cae de lado para desequilibrar y lanzar al uke. El Kodokan reconoce **16 técnicas**.

| [Hane-makikomi](hane-makikomi.md) | 跳巻込 | Enrollamiento de salto |
| [Soto-makikomi](soto-makikomi.md) | 外巻込 | Enrollamiento exterior |
| [Uchi-makikomi](uchi-makikomi.md) | 内巻込 | Enrollamiento interior |
| [Uki-waza](uki-waza.md) | 浮技 | Técnica flotante |
