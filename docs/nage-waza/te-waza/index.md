# Te-waza — 手技

Las **te-waza** (técnicas de mano) son proyecciones en las que los brazos y las manos constituyen la fuerza motriz principal del lanzamiento. El Kodokan reconoce **16 técnicas**.

| Técnica | Kanji | Traducción |
|---------|-------|------------|
| [Seoi-nage](seoi-nage.md) | 背負投 | Lanzamiento cargando sobre la espalda |
| [Ippon-seoi-nage](ippon-seoi-nage.md) | 本背負投 | Lanzamiento sobre la espalda con un solo brazo |
| [Seoi-otoshi](seoi-otoshi.md) | 背負落 | Derribo cargando sobre la espalda |
| [Tai-otoshi](tai-otoshi.md) | 体落 | Derribo de cuerpo |
| [Kata-guruma](kata-guruma.md) | 肩車 | Rueda de hombros |
| [Sukui-nage](sukui-nage.md) | 掬投 | Lanzamiento de cuchara |
| [Obi-otoshi](obi-otoshi.md) | 帯落 | Derribo del cinturón |
| [Uki-otoshi](uki-otoshi.md) | 浮落 | Derribo flotante |
| [Sumi-otoshi](sumi-otoshi.md) | 隅落 | Derribo de esquina |
| [Yama-arashi](yama-arashi.md) | 山嵐 | Tormenta de montaña |
| [Obi-tori-gaeshi](obi-tori-gaeshi.md) | 帯取返 | Contraataque agarrando el cinturón |
| [Morote-gari](morote-gari.md) | 双手刈 | Siega con ambas manos |
| [Kuchiki-taoshi](kuchiki-taoshi.md) | 朽木倒 | Derribo del árbol podrido |
