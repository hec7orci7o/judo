# Ma-sutemi-waza — 真捨身技

Las **ma-sutemi-waza** (técnicas de sacrificio frontal) son proyecciones en las que el tori cae deliberadamente hacia atrás para utilizar su propio peso y el desequilibrio del uke. El término *ma* indica que el sacrificio es directo, hacia atrás. El Kodokan reconoce **5 técnicas**.

| Técnica | Kanji | Traducción |
|---------|-------|------------|
| [Tomoe-nage](tomoe-nage.md) | 巴投 | Lanzamiento circular |
| [Sumi-gaeshi](sumi-gaeshi.md) | 隅返 | Inversión de esquina |
| [Hikikomi-gaeshi](hikikomi-gaeshi.md) | 引込返 | Inversión tirando hacia adentro |
| [Tawara-gaeshi](tawara-gaeshi.md) | 俵返 | Inversión del saco de arroz |
| [Ura-nage](ura-nage.md) | 裏投 | Lanzamiento por detrás |
